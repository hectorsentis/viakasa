Labeled text input for lead / contact forms. Gold focus ring, brand-caps label.

```jsx
<Input label="Nombre" placeholder="Tu nombre" />
<Input label="Email" type="email" iconLeft={<i data-lucide="mail" />} hint="Te responderemos en 24h" />
<Input label="Teléfono" error="Número no válido" />
```

Props: `label, type, placeholder, value, onChange, iconLeft, hint, error, disabled`.
